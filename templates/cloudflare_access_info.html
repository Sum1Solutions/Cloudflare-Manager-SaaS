Setup Instructions:
Enable Cloudflare Access

Create Application
Add a Self-hosted application for {{ request.headers.get('Host', 'your-domain.com') }}
Configure Identity Providers
Enable Google, GitHub, or One-time PIN authentication
Set Access Policy
Allow authenticated users to access this application
